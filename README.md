# Vendor_Performance_Analysis
An end-to-end Vendor Performance Analysis project using Python, MySQL, and Power BI. Includes data cleaning, EDA, SQL analysis, interactive dashboards, and business insights to evaluate vendor performance and support data-driven decision-making.

## Dataset

The dataset contains vendor-related business information, including purchasing, sales, inventory, and financial data.

The analysis focuses on:

* Vendor sales performance
* Purchase and inventory trends
* Profitability analysis
* Vendor contribution

## Project Workflow

### 1. Data Loading

* Import the dataset into Python
* Review data structure and column information
* Validate data quality

### 2. Exploratory Data Analysis (EDA)

* Analyze vendor performance metrics
* Identify missing values and outliers
* Explore sales, purchases, and inventory patterns
* Visualize trends and relationships

### 3. Data Cleaning

* Handle missing values
* Remove duplicate records
* Standardize data formats
* Prepare the dataset for analysis

### 4. SQL Analysis (MySQL)

* Import the cleaned dataset into MySQL
* Execute SQL queries to analyze:

  * Vendor-wise sales
  * Purchase performance
  * Profit margins
  * Inventory movement
  * Top and low-performing vendors
  * Business KPIs

### 5. Power BI Dashboard

* Develop an interactive dashboard featuring:

  * Vendor Performance Overview
  * Sales Analysis
  * Purchase Analysis
  * Profitability Metrics
  * Inventory Insights
  * Dynamic filters and slicers

### 6. Report Creation

* Document the analysis process
* Present key findings
* Provide business recommendations based on insights

### 7. Presentation

* Create a professional presentation using Gamma
* Summarize objectives, methodology, dashboard highlights, and recommendations
