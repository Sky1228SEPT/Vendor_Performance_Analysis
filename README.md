# Overview

Vendor performance plays a crucial role in supply chain efficiency and business profitability. This project analyzes vendor-related data to identify top-performing vendors, evaluate purchasing and sales trends, monitor inventory movement, and measure profitability.

Using **Python** for data preprocessing and EDA, **MySQL** for business analysis, and **Power BI** for visualization, the project transforms raw data into meaningful insights that support informed business decisions.

### Project Objectives

* Analyze vendor performance using business metrics.
* Clean and prepare raw data for analysis.
* Perform Exploratory Data Analysis (EDA).
* Write SQL queries to generate business insights.
* Build an interactive Power BI dashboard.
* Create a professional report and presentation.

# Project Workflow

### 1. Data Loading

* Import dataset into Python.
* Inspect data structure and data types.

### 2. Exploratory Data Analysis (EDA)

* Identify missing values.
* Detect outliers.
* Analyze trends and distributions.
* Visualize relationships.

### 3. Data Cleaning

* Handle missing values.
* Remove duplicate records.
* Correct inconsistent data.
* Prepare the dataset for analysis.

### 4. SQL Analysis (MySQL)

* Import cleaned data into MySQL.
* Execute SQL queries for:

  * Vendor Performance
  * Sales Analysis
  * Purchase Analysis
  * Profitability Analysis
  * Inventory Analysis
  * KPI Calculation

### 5. Power BI Dashboard

* Design interactive dashboards.
* Create KPIs and charts.
* Add filters and slicers.
* Visualize business performance.

### 6. Report Creation

* Document methodology.
* Summarize findings.
* Provide business recommendations.

### 7. Presentation

* Prepare a professional presentation using Gamma.
* Highlight objectives, analysis, dashboard, and conclusions.
